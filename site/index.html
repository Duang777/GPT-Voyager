CORE CAPABILITIES

你会高频用到的功能，已经组合好

会话索引与快速检索

自动扫描可见会话并建立本地索引，支持按标题、ID、标签、文件夹快速定位。

分类管理与批量操作

文件夹 + 标签双维管理，支持批量归类、批量加/去标签与一键撤销。

提示词库与变量模板

保存可复用 Prompt 片段，支持 `{{变量}}` 填充后直接插入输入框。

会话导出与数据备份

单条会话导出 Markdown/HTML，并可导出 JSON 备份进行跨设备迁移。

公式工作台

提取会话公式，支持 LaTeX 复制、MathML 复制与来源定位，适配论文和技术写作。

Mermaid 图表工作台

自动识别图表代码块，支持预览、源码复制和跳转定位，图形表达不再断链。

WORKFLOW

典型 4 步工作流

  1. 01

    抓取会话

    在 ChatGPT 页面自动扫描并同步当前会话列表，形成可搜索索引。

  2. 02

    分类沉淀

    按项目/主题打标签入文件夹，让长期对话从“历史记录”变成“知识资产”。

  3. 03

    复用加速

    在提示词库中复用模板和变量，一键插入，减少重复输入成本。

  4. 04

    导出归档

    将关键对话导出为 Markdown/HTML，配合 JSON 备份形成可迁移档案。

INSTALL

3 分钟上线到 GitHub Pages

项目已内置 Pages 自动部署工作流。你只需：

  1. 把代码推送到 GitHub 仓库（默认分支 `main`）。
  2. 在仓库 `Settings > Pages` 选择 `GitHub Actions` 作为来源。
  3. 等待 `Deploy static site to GitHub Pages` 工作流完成。

发布地址通常是： https://<你的 GitHub 用户名>.github.io/<仓库名>/
