# 功能记录

## 1. 元信息
- 功能 ID：FIX-007
- 功能名称：修复 PDF 导出公式渲染异常与清晰度问题
- 日期：2026-03-05
- 作者：Codex
- 分支：main

## 2. 概述
- 处理用户反馈“PDF 中公式显示异常（错位/发虚）”。修复内容聚焦导出渲染链路：提高渲染分辨率、避免极端离屏坐标引发排版精度问题、保留消息内容中的 `svg`，提升数学公式和图形内容在 PDF 中的稳定性与可读性。

## 3. 需求映射
- 对应 PRD 章节：`FR-06 导出/导入`
- 覆盖的用户场景：
  - 导出学术/技术会话时，公式在 PDF 中应保持可读和可辨识。
- 范围说明（In scope / Out of scope）：
  - In scope：当前会话 PDF 导出渲染质量修复。
  - Out of scope：矢量级公式原生排版（非截图流）与可编辑数学对象导出。

## 4. 设计
- 关键设计决策：
  - PDF 捕获容器从极大负坐标改为 `left: 0`，防止公式精细布局在截图时出现坐标精度误差。
  - 渲染比例从低倍率提高到 `2~3`，显著改善细线条公式清晰度。
  - 关闭 PDF 图片快速压缩，避免公式边缘锯齿和文字发糊。
  - 清理消息内容时不再删除全部 `svg`，避免误伤公式/图形渲染节点。
- 取舍说明：
  - 文件体积会增加，导出速度在超长会话下略慢，但换取更稳定的可读性。
- 权限/数据/隐私考虑：
  - 全程本地处理，无新增权限与外传数据。
