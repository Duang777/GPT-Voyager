# 前端重构：组织形式与视觉升级

## 1. 元信息
- 功能 ID：FEAT-008
- 功能名称：重构侧边栏组织形式并升级视觉风格
- 日期：2026-03-04
- 作者：Codex
- 分支：N/A（工作区直接更新）

## 2. 概述
针对“界面较丑、组织形式混乱”的反馈，完成一次结构与视觉并行重构。  
将原本长滚动单页改为分区导航，并升级主题、层级、卡片和状态呈现，使信息更聚焦、操作更清晰。

## 3. 需求映射
- 对应 PRD 章节：9（非功能需求-可维护性/体验）、10（技术方案）、15（DoD）
- 覆盖的用户场景：
  - 用户在不同任务（会话管理/提示词管理/系统设置）间快速切换。
  - 用户在高密度信息下仍能快速定位关键操作。
- 本次范围：
  - 分区导航：会话工作台 / 提示词库 / 设置中心
  - 会话工作台增加指标卡（可见数、索引数、文件夹数、标签数）
  - 全量视觉主题升级（色彩、边框、背景、按钮、动效）
  - 移动窄宽下响应式布局优化
- 本次不包含：
  - 独立路由与页面级拆分
  - 动态主题切换

## 4. 设计
- 关键设计决策：
  - 采用顶部分区导航减少“长页面滚动”造成的混乱。
  - 使用变量化主题色与渐变背景，统一视觉语言。
  - 用“高亮卡片 + 常规卡片”表达信息优先级。
  - 使用轻量动效（进场、悬停）增强反馈。
- 取舍说明：
  - 在保持现有功能完整的前提下，优先重构信息架构和交互路径。
- 权限/数据/隐私考虑：
  - 纯前端表现层变更，不涉及权限和数据外发。
