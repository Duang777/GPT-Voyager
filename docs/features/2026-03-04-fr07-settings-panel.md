# FR-07：基础设置面板

## 1. 元信息
- 功能 ID：FEAT-007
- 功能名称：实现基础设置面板与行为配置持久化（FR-07）
- 日期：2026-03-04
- 作者：Codex
- 分支：N/A（工作区直接更新）

## 2. 概述
在侧边栏新增“设置”模块，允许用户调整核心行为参数并持久化。当前支持自动扫描开关、扫描间隔、提示词插入模式、默认导出格式。

## 3. 需求映射
- 对应 PRD 章节：6.1（P0）、8（FR-07）、10（技术方案）、15（DoD）
- 覆盖的用户场景：
  - 用户希望按个人习惯调整插件行为。
  - 用户希望设置在刷新后保留。
- 本次范围：
  - 新增设置数据模型及本地存储
  - 新增设置 UI
  - 设置项接入业务行为
- 本次不包含：
  - 多设备同步设置
  - 设置导入导出

## 4. 设计
- 关键设计决策：
  - 设置键名：`gpt_voyager_settings_v1`。
  - 设置项：
    - `autoScanEnabled`
    - `scanIntervalSec`
    - `promptInsertMode`（append/replace）
    - `defaultExportFormat`（markdown/html）
  - 设置变化采用节流写入，降低频繁 `storage` I/O。
- 取舍说明：
  - 当前仅提供基础配置，优先稳定可用。
- 权限/数据/隐私考虑：
  - 设置仅保存在 `chrome.storage.local`，无外发请求。
