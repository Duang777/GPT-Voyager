# 功能记录

## 1. 元信息
- 功能 ID：FEAT-052 / UI-021
- 功能名称：新增“极简紧凑”卡片密度（隐藏备注输入，点击卡片展开）
- 日期：2026-03-05
- 作者：Codex
- 分支：main

## 2. 概述
- 在现有“标准/紧凑”基础上新增第三档密度“极简紧凑”。  
- 该模式默认隐藏会话备注输入框，仅在点击卡片空白区域时展开备注编辑，再次点击可收起。

## 3. 需求映射
- 对应 PRD 章节：`FR-02 会话索引`
- 覆盖的用户场景：
  - 用户希望在高密度列表中减少视觉噪声，仅在需要时再编辑备注。
- 范围说明（In scope / Out of scope）：
  - In scope：密度选项新增、卡片渲染与交互、紧凑样式联动、虚拟滚动行高联动。
  - Out of scope：会话备注数据模型与存储逻辑。

## 4. 设计
- 关键设计决策：
  - 新增密度枚举值 `minimal`，前台文案为“极简紧凑”。
  - 极简模式下：
    - 默认显示“备注预览/点击提示”占位行；
    - 点击卡片空白区域展开备注输入；
    - 点击交互控件（按钮/下拉/标签/复选框）不触发展开。
  - 虚拟滚动新增极简行高常量，保障滚动窗口估算稳定。
- 取舍说明：
  - 采用“单卡片当前展开”的轻交互，避免密度模式下同时展开多卡导致布局混乱。

## 5. 实现
- 主要变更文件：
  - `src/content/settingsStore.ts`
  - `src/content/App.tsx`
  - `src/content/index.tsx`
  - `PRD.md`
- 核心逻辑：
  - `settingsStore.ts`：扩展 `ConversationCardDensity` 与解析逻辑支持 `minimal`。
  - `App.tsx`：
    - 密度选项新增“极简紧凑”。
    - 增加 `minimalExpandedConversationId` 状态控制卡片展开。
    - 按密度选择不同虚拟行高，新增极简行高常量。
    - 会话卡片渲染分支：极简模式默认隐藏输入，按需展开输入。
  - `index.tsx`：新增 `.gv-list-minimal` 系列样式与备注占位行样式。
- 数据模型/存储变更：
  - 无新增字段，仅扩展枚举值。
- 配置/Manifest 变更：
  - 无。

## 6. 验证
- 自动化测试：
  - `npm run typecheck` 通过。
  - `npm run build` 通过。
- 手工验证步骤：
  1. 在会话索引中切换“卡片密度=极简紧凑”。
  2. 确认备注输入默认隐藏，仅显示提示/预览行。
  3. 点击卡片空白区域，确认备注输入展开；再次点击收起。
  4. 点击按钮、下拉、标签等控件时，不应误触发展开。
- 测试结果摘要：
  - 极简模式交互生效，备注编辑按需展开，整体密度更轻。

## 7. 风险与回滚
- 已知风险：
  - 卡片点击区域与交互控件边界依赖 `closest` 选择器，后续控件结构变化需同步维护。
- 回滚策略：
  - 回滚新增密度枚举与极简渲染分支样式。
